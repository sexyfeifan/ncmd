# 网易云音乐下载器使用说明

## 快速开始

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行程序：
   ```bash
   python main.py
   ```

## 详细使用步骤

### 1. 登录账号
首次使用需要登录网易云音乐账号：
- 点击"登录网易云"按钮
- 程序会自动打开浏览器
- 在浏览器中登录您的网易云音乐账号
- 登录成功后Cookie会自动保存

### 2. 解析歌单
- 复制网易云音乐歌单的URL
- 粘贴到"歌单 URL"输入框中
- 点击"解析歌单"按钮
- 等待程序获取歌单信息

### 3. 选择歌曲
- 在歌曲列表中勾选想要下载的歌曲
- 可以使用"全选"或"取消全选"按钮
- 已下载的歌曲会自动识别并标记

### 4. 设置下载选项
- **音质选择**：选择下载音质（标准/较高/无损等）
- **命名格式**：选择文件命名方式
- **排序方式**：选择歌曲显示顺序
- **并发数**：设置同时下载的歌曲数量（1-5）
- **下载歌词**：是否同时下载歌词文件
- **按专辑分组**：是否按专辑创建子文件夹
- **创建歌单文件夹**：是否为歌单创建单独文件夹

### 5. 开始下载
- 点击"开始下载"按钮
- 查看下载进度条和速度显示
- 可以随时暂停、继续或取消下载

## 音质说明

支持以下音质选择：
- 标准：约 128kbps MP3
- 极高：约 320kbps MP3
- 无损：FLAC格式，约 1000kbps
- 高清臻音：特殊编码的高质量音频
- 超清母带：专业级别的高保真音频
- 沉浸环绕声：3D环绕声效果音频

注意：部分高品质音质需要网易云音乐会员权限。

## 智能音质降级

当选择的音质不可用时（如会员限制），程序会自动降级到可用的较低音质：
1. 沉浸环绕声 (sky)
2. 超清母带 (jymaster)
3. 高清臻音 (jyeffect)
4. 无损 (lossless)
5. 极高 (exhigh)
6. 标准 (standard)

## 文件命名格式

支持多种文件命名格式：
- 歌曲名 - 艺术家.mp3
- 艺术家 - 歌曲名.mp3
- 歌曲名 - 艺术家 [音质].mp3
- 艺术家 - 歌曲名 [音质].mp3

## 下载目录

默认下载目录为系统的Music文件夹，可以点击"选择下载目录"按钮更改。

## 常见问题

### 1. 下载速度慢
- 调整并发数（建议3-5）
- 选择较低音质
- 确保网络连接稳定

### 2. 登录失败
- 确保已安装Chrome浏览器
- 运行：`pip install --upgrade selenium`
- 重启程序后重试

### 3. 下载失败
- 检查是否为会员限制
- 程序会自动降级音质重试
- 查看日志文件获取详细信息
