# 网易云音乐下载器 NeteaseMusicDownloader v2.4.0

## 简介

网易云音乐下载器是一款基于Python开发的网易云音乐歌曲下载工具，支持完整的歌曲搜索、歌手页面跳转、专辑页面查看、批量下载歌曲等功能。该工具具有友好的图形界面，支持多种音质选择和智能音质降级机制。

## 功能特性

- 🎵 完整的歌曲搜索功能
- 👤 歌手页面跳转
- 💿 专辑页面查看
- 📥 批量下载歌曲
- 🎵 多种音质选择（标准/较高/无损/Hi-Res等）
- 🏷️ 智能音质降级（当选择音质不可用时自动降级）
- 🖼️ 完善的元数据嵌入（包括FLAC封面）
- 📂 按专辑分组下载
- 🔀 多种命名格式
- ⚡ 可调节的并发下载（1-5个并发）
- 📊 实时下载进度显示
- 🛑 暂停/继续/取消功能
- 📂 已下载歌曲识别

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 图形界面版本（推荐）
```bash
python main.py
```

### 命令行版本
```bash
python cli_main.py
```

## 构建为独立应用

### macOS
```bash
python build.py
```

## 系统要求

- Python 3.6+
- macOS推荐（构建脚本主要针对macOS优化）

## 注意事项

1. 首次使用需要登录网易云音乐账号
2. 部分高品质音质需要会员权限
3. 下载的音乐仅供个人学习和欣赏，请尊重版权
4. 请遵守相关法律法规，勿用于商业用途

## 技术架构

- **核心框架**：Python 3.6+
- **GUI框架**：Flet
- **网络请求**：requests
- **API接口**：pyncm（网易云音乐API封装）
- **音频处理**：mutagen（元数据处理）
- **图像处理**：Pillow（封面处理）
- **浏览器自动化**：selenium（登录功能）
